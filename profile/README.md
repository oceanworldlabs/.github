# Ocean World Labs

**Frontier AI for the Ocean.**

We are building the simulation and intelligence stack for the ocean—from physics to foundation models to embodied systems.

**OceanScale** is the foundation: a GPU-native virtual ocean for developing and evaluating underwater robots, with BlueROV2 Heavy as the reference system.
